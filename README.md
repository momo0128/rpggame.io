# RPG GAME

## Overview
ターミナル上で動く、RPGゲームです。rubyの練習用に作成しました。

## Features
* 敵を倒していくゲーム
* ガチャを回して自分の武器や武器のレベル、所持金を得ることができる。
* ミニゲームでは、自分のステータスを上げられるチャンスを得られる。
* 途中でゲームを終了しても続きから始められる。
